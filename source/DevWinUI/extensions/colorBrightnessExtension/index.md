---
title: ColorBrightnessExtension
---

MarkupExtension to that can be used to brighten (positive) or darken (negative) a Color.

# Property

|Name|
|-|
|CorrectionFactor|
|Color|

# Example

```xml
<SolidColorBrush Color = "{dev:ColorBrightness Color=Red, CorrectionFactor=-0.3}" />
```
