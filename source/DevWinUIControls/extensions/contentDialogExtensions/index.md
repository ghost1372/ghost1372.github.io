# Methods

|Name|
|-|
|ShowAsWindowAsync|

# Example

```cs
ContentDialog dialog = new()
{
    Title = "Title",
    Content = "Content",
    PrimaryButtonText = "Primary",
    SecondaryButtonText = "Secondary",
    CloseButtonText = "Close"
};

await dialog.ShowAsWindowAsync();
```
