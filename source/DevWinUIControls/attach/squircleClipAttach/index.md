---
title: SquircleClipAttach
---

CornerSmoothing is an attached property that applies a squircle-style corner shape to any UI element by generating a composition clip with non-linear smoothing. It allows you to go beyond normal rounded rectangles and achieve the softer, modern corner style.

# Property

|Name|
|-|
|CornerRadius|
|CornerSmoothing|

```xml
<Border Background="#56ccf2" Width="200" Height="100" dev:SquircleClipAttach.CornerSmoothing="0.8"/>

```
