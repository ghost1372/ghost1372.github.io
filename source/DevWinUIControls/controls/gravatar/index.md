---
title: Gravatar
---

A way to implement random avatars for users.


# Attributes
|Property|Description|Default Value|Remarks|
|-|-|-|-|
|Generator|Avatar Generator|GithubGravatarGenerator||
|Id|UserId|||
|Source|User Profile||||

# Example

```xml
<dev:Gravatar Id="User1"/>
```
