# Property

|Name|
|-|
|Header|
|MainContent|
|SideContent|

# Example

```xml
 <dev:SamplePanel Header="Header">
    <dev:SamplePanel.MainContent>
        <TextBlock Text="Main Content" />
    </dev:SamplePanel.MainContent>
    <dev:SamplePanel.SideContent>
        <TextBlock Text="Side Content" />
    </dev:SamplePanel.SideContent>
</dev:SamplePanel>
```
