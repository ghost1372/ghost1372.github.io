# Attributes

| Name |
|-|
|TitleContent|
|Title|
|Content|
|DividerVisibility|

# Example

```xml
<dev:Card Title="Title">
    <dev:Card.TitleContent>
        <Button Margin="0,0,10,0"
                Content="Title Content" />
    </dev:Card.TitleContent>
    <dev:Card.Content>
        <Button Content="Content" />
    </dev:Card.Content>
</dev:Card>
```
