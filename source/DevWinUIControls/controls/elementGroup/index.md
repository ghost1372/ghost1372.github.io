# Attributes
|Property|
|-|
|Orientation|

# Example

```xml
<dev:ElementGroup>
    <Button HorizontalAlignment="Stretch"
            Content="Button" />
    <Button HorizontalAlignment="Stretch"
            Content="Button" />
    <Button HorizontalAlignment="Stretch"
            Content="Button" />
    <Button HorizontalAlignment="Stretch"
            Content="Button" />
</dev:ElementGroup>
```
