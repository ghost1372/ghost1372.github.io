# Attributes
|Property|
|-|
|Side1|
|Side2|
|IsActive|

# Example

```xml
<dev:FlipSide>
    <dev:FlipSide.Side1>
        <Grid Background="Red">
            <TextBlock HorizontalAlignment="Center"
                       VerticalAlignment="Center"
                       Text="Side 1" />
        </Grid>
    </wuc:FlipSide.Side1>
    <dev:FlipSide.Side2>
        <Grid Background="Blue">
            <TextBlock HorizontalAlignment="Center"
                       VerticalAlignment="Center"
                       Text="Side 2" />
        </Grid>
    </wuc:FlipSide.Side2>
</wuc:FlipSide>
```
