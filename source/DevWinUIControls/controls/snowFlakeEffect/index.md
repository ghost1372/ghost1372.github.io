---
title: SnowFlakeEffect
---

A visual effect that simulates falling snowflakes, with customizable particle count.

# Attributes
|Property|
|-|
|AutoStart|
|FlakeCount|

# Methods
|Property|
|-|
|Start|
|Stop|

# Example

```xml
<dev:SnowFlakeEffect FlakeCount="100" />

```
