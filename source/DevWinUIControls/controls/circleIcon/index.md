---
title: CircleIcon
---

CircleIcon is a circular control with customizable border, gradient background, and optional content.

# Property

|Name|
|-|
|BorderStrokeThickness|
|Content|
|BorderStrokeColor|
|BackgroundGradientStops|
|ClearColor|

# Example

```xml
<dev:CircleIcon Content="D"/>
```
